node-Windows 常用命令

1. 创建Windows服务

使用以下命令可以将node.js应用程序作为Windows服务运行： 《Node.js官网》 《直接下载》


var Service = require('node-windows').Service;

var svc = new Service({
  name: 'MyService',
  description: 'This is my node.js service.',
  script: 'C:\\path\\to\\app.js'
});

svc.install();

2. 启动服务

安装服务后可以通过以下命令启动服务：

svc.start();

3. 停止服务

要停止正在运行的服务，可以使用：

svc.stop();

4. 卸载服务

若需要卸载服务，使用以下命令：

svc.uninstall();

5. 监听服务事件

可以通过以下方式监听服务的安装、启动、停止事件：


svc.on('install', function() {
  svc.start();
});

svc.on('uninstall', function() {
  console.log('Service uninstalled');
});

svc.on('start', function() {
  console.log('Service started');
});

6. 获取服务状态

通过 isInstalled 方法可以检查服务是否已安装：

console.log(svc.exists);

7. 检测依赖更新

在安装前，先检测依赖包的更新情况：

npm outdated

8. 安装依赖

可以使用以下命令安装所有依赖：

npm install

9. 更新依赖

如果需要更新依赖包到最新版本，可以使用：

npm update

10. 删除依赖

如果不再需要某个依赖包，可以使用：

npm uninstall <依赖包名>

11. 全局安装包

使用 -g 参数来全局安装某个包：

npm install -g <包名>

12. 查看全局安装的包

列出所有全局安装的依赖包：

npm list -g --depth=0

13. 修复依赖包

如果依赖包安装出现问题，可以尝试使用以下命令修复：

npm audit fix

14. 启动项目

启动项目，通常使用以下命令：

npm start

15. 运行项目

使用 node 运行项目的入口文件：

node app.js

16. 使用PM2启动项目

使用PM2进行进程管理：

pm2 start app.js

17. 重新启动项目

重启项目：

pm2 restart app.js

18. 停止项目

停止PM2管理的项目：

pm2 stop app.js
